JAVASCRIPT TİP DÖNÜŞÜMLERİ

Javascript tip dönüşümlerini otomatik olarak kendisi yapabileceği gibi Number ile number türüne, String ile string türüne, Boolean ile boolean türüne dünüşüm yapılabilir. Javascript değişken tipleri;
1- string
2- number
3- boolean
4- object
5- function
Javascript'te 3 farklı nesne türü vardır;
1- object
2- date
3- array
Javascript'te değer içermeyen 2 değişken türü vardır;
1- null
2- undefined

Javascript değişkenleri yeni bir değişkene veya başka bir değişken türüne dönüştürmek için ;
Javascript tip dönüşüm fonksiyonları veya javascript ile otomatik olarak dönüştürebiliriz.

Sayı Veri Türünü Metin Veri Türüne Dönüştürme

Sayıları metin veri türüne dönüştürmek için String fonksiyonu kullanılabilir.

  var xSayisi = 1453;
  alert(String(xSayisi));
  alert(String(123));
  alert(String(100 + 23));

Sayı veri türünün toString fonksiyonu sayıları metin veri türüne dönüştürür.

  var xSayisi = 1453;
  alert(xSayisi.toString());
  alert((123).toString());
  alert((100 + 23).toString());

Mantık Veri Türünü Metin Veri Türüne Dönüştürme

Mantık veri türünü metin veri türüne dönüştürmek için String fonksiyonu kullanılabilir.

  alert(String(false));
  alert(String(true));

Mantık veri türünü metin veri türüne dönüştürmek için toString fonksiyonu kullanılabilir.

  alert(false.toString());
  alert(true.toString());

Tarih Veri Türünü Metin Veri Türüne Dönüştürme

Tarih veri türünü metin veri türüne dönüştürmek için String fonksiyonu kullanılabilir.

alert(Date());

Tarih veri türünü metin veri türüne dönüştürmek için toString fonksiyonu kullanılabilir.

alert(Date().toString());

Metin Veri Türünü Sayı Veri Türüne Dönüştürme

Metin veri türünü sayı veri türüne dönüştürmek için Number fonksiyonu kullanılabilir. Metin veri türü değeri boş ise 0'a, diğer tüm metin türü değerleri Nan değerine dönüşür.

  alert(Number("3.14"));
  alert(Number(" "));
  alert(Number(""));
  alert(Number("99 98"));

Mantık Veri Türünü Sayı Veri Türüne Dönüştürme

Mantık veri türünü sayı veri türüne dönüştürmek için Number fonksiyonu Kullanılabilir.

  alert(Number(false));
  alert(Number(true));

Tarih Veri Türünü Sayı Veri Türüne Dönüştürme

Tarih veri türünü sayı veri türüne dönüştürmek için Number fonksiyonu kullanılabilir.

var tarih = new Date();
  alert(Number(new Date()));
  alert(Number(tarih));

Tarih veri türünün getTime fonksiyonu tarih veri türünü sayı veri türüne dönüştürür.

var tarih = new Date();
  alert(tarih.getTime());

Otomatik Tip Dönüşümü

Javascript farklı veri türleri üzerinden işlem yapıldığında veri türünü otomatik olarak doğru veri türüne dönüştürmeye çalışır. Ancak sonuç her zaman doğru olmayabilir.

  alert(5 + null);    // 5
  alert("5" + null);  // "5null"
  alert("5" + 2);     // 52
  alert("5" - 2);     // 3
  alert("5" * "2");   // 10

Otomatik Metin Veri Türüne Dönüştürme

Javascript bir nesneyi veya değişkeni ekrana yazdırmaya calıştığınızda toString fonksiyonunu otomatik olarak çalıştırır.

  alert({name:"Fjohn"});  // [object Object]
  alert([1,2,3,4]);       // 1,2,3,4
  alert(new Date(0));     // Thu Jan 01 1970 03:00:00 GMT+0300

Sayı mantık veri türünü de dönüştürür ancak bu dönüşüm pek fark edilmez.

  alert(123);    // 123
  alert(true);   // true
  alert(false);  // false
